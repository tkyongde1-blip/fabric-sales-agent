# 标准话术库

## 一、抖音/视频号评论区回复

### 询价类
```
亲，私信您了哈～我们的面料都是厂价直发，加个微信发您详细报价和样品📦
```

### 产品类
```
这个款我们有现货哦～牛奶丝/罗纹/四面弹都有，私信我发您产品目录和报价
```

### 意向强
```
好的亲，马上安排销售和您对接！麻烦通过一下好友申请，发您详细资料和报价～
```

```

我们做牛奶丝、罗纹、四面弹这些针织面料，厂价直发，没有中间商。
方便加个微信吗？我发您产品目录和报价单～
```

### 引导加微信
```
这是我的微信号：XXXXX，加我备注需求，我给您发详细报价和样品图。
或者您方便的话留个电话，我直接打给您说～
```

### 已加微信未通过
```
您好，我是刚才和您聊面料的小X。
白天在仓库可能回复慢，看到消息第一时间回您哈～
要什么面料随时说！
```

## 三、微信销售话术

### 3.1 新客户欢迎
```
X总您好！我是XX针织厂的小X。
我们厂专业做针织面料 X 年了，主营牛奶丝、罗纹布、四面弹、汗布、摇粒绒。
有展厅和仓库，支持看货打样，广东省内次日达，省外3天左右。
请问您这次主要想看什么面料？做什么款式？
```

### 3.2 需求挖掘
```
好的，帮您记一下：
- 产品: XX
- 数量: XX 米/公斤
- 用途: XX
- 交期: XX前要

我先给您算个报价，马上发给您！
```

### 3.2 报价发送
```
X总，这是刚才说的 XX 面料报价单：
- 品名: XX
- 规格: XX
- 单价: XX元/米
- 起订量: XX米
- 交期: 现货/预定X天

我们是厂价直发，同样的品质您可以去比，性价比绝对高。
现货充足，随时可以发货！
```

### 3.3 应对比价
```
X总，价格是一分钱一分货的。
我们做的是 A 品足米足秤，不偷工减料。
市面上比我们便宜的肯定有，但您得看质量、看服务、看售后。
您要是方便，我先寄个样品给您看，满意了再下单，不耽误您时间！
```

### 3.4 应对砍价
```
X总，我给您报的已经是实价了。
我们是工厂直营，没有中间环节，价格透明。
量大我们可以优惠一点点，但降价肯定会降品质，得不偿失。
```

### 3.5 客户说"考虑"
```
理解，X总您先考虑。
不过这款面料最近走得很快，库存不多了。
您方便的话先定个小批量试试，用好了再续，也不耽误您的交期。
或者您有什么顾虑，直接跟我说，我帮您解决。
```

### 3.6 成交确认
```
好嘞！那帮您确认一下订单：
1. 产品: XX面料 XX米
2. 单价: XX元/米
3. 总金额: XX元（现金/转账）
4. 交期: XX日发货
5. 地址: XXXXXXXXX

确认没问题的话，我安排仓库备货，发货前拍照给您确认！
```

## 四、跟进话术

### 4.1 报价后跟进（24H）
```
X总，昨天发您的报价看了吗？
这款面料现在库存充足，质量您放心，随时可以发货。
您看要不要先安排个小单试试？
```

### 4.2 B 类客户跟进（3天）
```
X总，最近生意怎么样？
我们新到了一批 XX 面料，花色很不错，发您看看。
有需要随时联系我哈～
```

### 4.3 C 类客户跟进（7天）
```
X总，天凉了注意身体。
最近面料行情有点波动，有几款在做活动，有需要随时问我～
```

### 4.4 沉睡客户唤醒（15天）
```
X总，好久没联系了，最近还在做服装吗？
我们最近出了几个新花色，质量好价格也合适，发您看看？
方便的话记得回复我一下哈～
```

## 五、售后话术

### 5.1 发货通知
```
X总，您的货已经发出啦！
物流单号: XXXXXXXX
预计 X 天到货，到了麻烦确认一下质量，
有任何问题随时找我！
```

### 5.2 收货回访（3天）
```
X总，货收到了吗？质量还满意吗？
有什么问题随时和我说，不满意包退换！
满意的话麻烦介绍给同行，提您名字有优惠～
```

### 5.3 质量问题
```
X总，非常抱歉出现这个问题！
您拍个照片/视频给我，我看看具体情况。
是我们的问题一定负责到底，给您补发或退款。
```

## 六、拒绝/终结话术

### 6.1 赊账要求
```
X总，不好意思，我们一直都是现金交易的，不做月结。
您可以先拿小批量试试，货到满意了再复购。
长期合作了可以谈，但第一次只能现金哈～
```

### 6.2 无理砍价
```
X总，这个价格确实做不了。
我们的面料是足米足秤的 A 品，成本在那里。
您也可以去别家看看，觉得合适了再来找我。
```

### 6.3 非客户
```
好的，打扰了，以后有需要随时找我哈～
祝您生意兴隆！
```
