# 客户分类规则

## 三级分类体系

### A 类客户（重点跟进 ≥ 80 分）

**定义**: 意向明确、需求匹配、决策周期短的客户

**评分条件**（满足 3 条以上）：
- [ ] 明确说出产品名称（牛奶丝/罗纹/四面弹等）
- [ ] 有明确数量需求（多少米/多少公斤）
- [ ] 有明确时间计划（这周要/月底要/现货）
- [ ] 主动询问价格并接受合理价格
- [ ] 主动留下联系方式（电话/微信）
- [ ] 已加微信并回复信息
- [ ] 愿意打样或看样品
- [ ] 有相关行业背景（服装厂/贸易公司）

**跟进策略**:
- 24 小时内电话联系
- 当天发送报价单
- 优先安排样品
- 标记为"重点跟进"

**典型话术标签**:
- "我要 XX 面料，什么价格？"
- "能发样品吗？"
- "我现在就要，有现货吗？"
- "我在 XX（城市），能发货吗？"

---

### B 类客户（持续跟进 50-79 分）

**定义**: 有意向但信息不完整，需要持续培育的客户

**评分条件**（满足 2 条以上）：
- [ ] 问了价格但没具体数量
- [ ] 问了产品但没有明确品种
- [ ] 说"考虑考虑"、"再看看"
- [ ] 加了微信但没回复
- [ ] 问了样品但没有明确意向
- [ ] 提到在和其他家比较
- [ ] 问的是低毛利产品

**跟进策略**:
- 3 天内微信跟进
- 提供产品资料/样品
- 了解实际需求
- 不要频繁推送

**典型话术标签**:
- "这个什么价格？"（不说不量）
- "有样品吗？"（不进一步）
- "我考虑一下"
- "你家质量怎么样？"

---

### C 类客户（观察维护 0-49 分）

**定义**: 意向低、质量存疑、价格敏感型客户

**评分条件**（满足 1 条以上）：
- [ ] 只问价格不问质量
- [ ] 砍价幅度超过 20%
- [ ] 明确表示要赊账/月结
- [ ] 恶意比价、反复砍价
- [ ] 问的问题与面料无关
- [ ] 长时间不回复（30 天以上）
- [ ] 要求先货后款

**跟进策略**:
- 7-15 天一次简单跟进
- 不主动提供报价
- 朋友圈/视频号内容触达
- 标准回复即可

**典型话术标签**:
- "最低多少钱？"
- "能月结吗？"
- "XX 家比你便宜"
- "能先发点试试吗？"（不提付款）

## 自动评分计算公式

```
总分 = 意向分(0-40) + 决策分(0-30) + 匹配分(0-30)

意向分:
  - 明确产品需求: +20
  - 询问具体参数: +10
  - 要求看样品: +10

决策分:
  - 有明确时间: +15
  - 有数量需求: +10
  - 留联系方式: +5

匹配分:
  - 需求匹配库存: +15
  - 预算匹配报价: +10
  - 地理位置可发货: +5

降分项:
  - 赊账要求: -20
  - 无理砍价: -15
  - 非目标客户: -30
```

## 动态调整规则

1. **升级**: B 类客户主动询问并表现出明确意向 → 升级为 A 类
2. **降级**:
   - A 类 7 天未成交且回复冷淡 → 降为 B 类
   - B 类 14 天未回复 → 降为 C 类
3. **唤醒**:
   - C 类/沉睡客户再次询价 → 升级评估
   - 复购客户自动升级为 A 类
4. **黑名单**:
   - 恶意骚扰、欺诈、同行刺探 → 直接拉黑
