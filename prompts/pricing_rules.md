# 报价规则

## 一、成本核算

### 成本构成
```
面料成本 = 原材料成本 + 织造费 + 染整费 + 运费 + 包装费
```

| 产品 | 规格 | 成本价(元/m) | 标准报价(元/m) | 最低价(元/m) |
|-----|------|------------|--------------|------------|
| 牛奶丝 | 150cm 180g/m² | 18 | 24 | 22 |
| 罗纹布 | 150cm 220g/m² | 22 | 29 | 26 |
| 四面弹 | 150cm 200g/m² | 28 | 36 | 33 |
| 汗布 | 180cm 160g/m² | 15 | 20 | 18 |

> 注：以上价格为示例，实际价格按市场行情浮动，需定期更新。

## 二、报价规则

### 2.1 新客户报价
- **首次报价**: 成本价 × 1.3（加价 30%）
- **底价**: 成本价 × 1.15（最低利润 15%）
- **让步空间**: 5-10%，分两次让价

### 2.2 批量折扣
| 订单量 | 折扣 |
|-------|------|
| < 500m | 标准价 |
| 500-999m | 标准价 × 0.98 |
| 1000-2999m | 标准价 × 0.95 |
| 3000m+ | 标准价 × 0.92（需审批）|

### 2.3 老客户定价
- **复购客户**: 标准价 × 0.95（老客户价）
- **月结客户**（不适用于本业务，仅作参考）
- **VIP 客户**: 月采购 10000m+ 享受 VIP 价，需申请

### 2.4 急单加价
- 正常交期（7-15天）: 标准价
- 加急（3-5天）: 标准价 × 1.05
- 特急（1-2天）: 标准价 × 1.10

## 三、报价策略

### 3.1 分步报价法
```
第1步: 报标准价（留空间）
第2步: 客户还价 → 降3-5%（"我去申请一下"）
第3步: 客户继续还价 → 降2-3%（"这是底线了"）
第4步: 底线价 → 不降了（"再降就亏本了"）
```

### 3.2 报价附带价值
报价时同步说明附加价值，增强说服力:
- 免费样品
- 质量保证（可退货）
- 稳定供应
- 长期合作优惠

### 3.3 价格谈判底线
```
绝不触碰的底线:
1. 不低于成本价 × 1.15
2. 绝不赊账
3. 不包运费（除非大单）
4. 不送超出规格的样品数量
```

## 四、自动报价逻辑

### 4.1 输入参数
- 产品名称
- 规格（门幅、克重）
- 数量
- 是否老客户
- 是否加急
- 运费承担

### 4.2 计算流程
```
1. 查询产品基础成本价
2. 根据数量计算折扣系数
3. 根据客户类型计算价格系数
4. 根据交期计算加价系数
5. 计算最终报价 = 成本价 × 加价系数 × 折扣 × 客户系数 × 紧急系数
6. 校验是否 ≥ 最低利润价
7. 生成报价单
```

### 4.3 报价单包含信息
- 报价编号（自动生成）
- 报价日期
- 客户名称
- 产品明细（名称、规格、单价、数量）
- 总金额
- 交期
- 报价有效期（7天）
- 付款方式（现金/转账）
