# 销售标准作业流程 (SOP)

## 概述

本 SOP 适用于所有面料销售环节。所有员工必须严格按照此流程执行，确保服务质量和成交率。

## 一、获客阶段

### 1.1 公域获客（抖音/视频号）

**执行人**: AcquisitionAgent / 运营人员

**步骤**:
1. 每天 9:00、14:00、20:00 三次扫描账号评论区
2. 识别以下关键词评论：
   - 询价类："多少钱"、"价格"、"怎么卖"、"怎么买"
   - 产品类："牛奶丝"、"罗纹"、"四面弹"、"针织布"、"面料"
   - 意向类："样品"、"拿货"、"批发"、"订货"
3. 对含关键词的评论回复引导私信
4. 私信回复标准话术（详见 standard_scripts.md）
5. 获取微信号或电话后，在 CRM 创建线索

### 1.2 私信跟进

**执行人**: 销售 Agent / 销售员

**步骤**:
1. 收到私信后 15 分钟内首次回复
2. 3 条私信内引导加微信
3. 获得微信号后，CRM 标注"已加微信"
4. 未回复的 2 小时后再次发送

### 1.3 微信添加

**执行人**: 销售人员

**步骤**:
1. 添加时备注：城市+产品（如"广州-罗纹"）
2. 添加成功后发送欢迎话术
3. 标签分组：A/B/C + 产品分类
4. 记录首次沟通摘要到 CRM

---

## 二、需求挖掘阶段

### 2.1 首次沟通

**话术目标**: 5 个必问信息
1. 要什么产品？（牛奶丝/罗纹/四面弹/其他）
2. 做什么用途？（T恤/卫衣/裤子/家居服）
3. 要多少量？（米数/公斤）
4. 什么时候要？（现货/预定/不急）
5. 之前在哪拿货？（竞争对手分析）

### 2.2 需求录入

**操作**: 沟通结束后 10 分钟内录入 CRM
**必填字段**:
- 产品需求
- 预估数量
- 期望交期
- 预算范围
- 客户来源

---

## 三、报价阶段

### 3.1 报价原则

1. 首次报价在成本价上加价 30%（可谈空间 5-10%）
2. 批量订单（≥1000m）可给折扣价
3. 老客户复购给老客户价（比新客户低 3-5%）
4. 现金订单优先报实价
5. **绝不赊账，不议价底线**

### 3.2 报价流程

```
客户询价 → 查询产品目录 → 成本核算 → 加价计算 → 生成报价单 → 发送客户
```

### 3.3 发送报价后

1. 报价发送后 24 小时未回复 → 跟进
2. 客户对比价格 → 说明质量优势
3. 客户要求降价 → 最多让步 5-10%

---

## 四、跟进阶段

### 4.1 跟进频率

| 客户类别 | 跟进频率 | 跟进方式 |
|---------|---------|---------|
| A 类    | 每天 1 次 | 电话 + 微信 |
| B 类    | 每 3 天 1 次 | 微信为主 |
| C 类    | 每 7 天 1 次 | 微信 |
| 沉睡    | 每 15 天 1 次 | 朋友圈互动+群发 |

### 4.2 跟进内容策略

- **第1次跟进**: 报价 + 产品推荐
- **第2次跟进**: 样品方案 + 成功案例
- **第3次跟进**: 限时优惠 / 库存提醒
- **长期跟进**: 行业资讯 + 新品发布

### 4.3 跟进记录

**每次跟进后必须记录**:
1. 跟进时间
2. 客户反馈
3. 客户情绪（正面/负面/中性）
4. 下一步计划
5. 是否更新客户分类

---

## 五、成交阶段

### 5.1 成交确认

1. 确认订单明细：产品、数量、规格、价格
2. 确认交期
3. 确认付款方式（仅限现金/转账）
4. 确认发货地址

### 5.2 发货流程

1. 打印发货单
2. 仓库备货拍照发客户确认
3. 物流发出后发单号给客户
4. 客户确认收货后归档

### 5.3 售后

1. 确认收货后 3 天回访
2. 质量问题按标准退换流程
3. 满意度调查录入 CRM

---

## 六、淘汰与转介绍

### 6.1 客户降级/淘汰

- 30 天未回复 → 从 A/B 类降为 C 类
- 60 天未回复 → 标记为"沉睡客户"
- 180 天未回复 → 标记为"流失客户"
- 无理纠缠、恶意砍价 → 直接标注"黑名单"

### 6.2 转介绍激励

- 老客户介绍成交 → 返现或礼品
- 转介绍信息自动录入 CRM
