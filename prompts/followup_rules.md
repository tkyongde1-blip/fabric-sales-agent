# 跟进规则

## 一、跟进总则

1. **及时性**: 线索产生后 15 分钟内首次联系
2. **频率控制**: 按客户分类严格执行频率
3. **内容定制**: 每次跟进内容不同，避免模板化
4. **记录完整**: 每次跟进必须记录反馈
5. **动态调整**: 根据客户反馈实时调整分类

---

## 二、自动化跟进触发规则

### 2.1 事件驱动跟进

| 触发事件 | 响应动作 | 时限 |
|---------|---------|------|
| 新客户导入 | 发送欢迎话术 + 产品目录 | 即时 |
| 报价已发送 | 24H 后自动提醒跟进 | 24H |
| 客户回复 | 提取意图 + 自动回复建议 | 即时 |
| 客户长时间未回复 | 系统提示跟进 | 按类别周期 |
| 客户提到竞品 | 记录到 CRM + 生成应对策略 | 即时 |
| 客户砍价超限 | 提示销售人员 | 即时 |
| 客户要赊账 | 触发拒绝话术 | 即时 |

### 2.2 时间驱动跟进

| 类别 | 跟进周期 | 具体规则 |
|------|---------|---------|
| A 类 | 每天 1 次 | 10:00 自动生成跟进提醒 |
| B 类 | 每 3 天 1 次 | 10:00 自动生成跟进提醒 |
| C 类 | 每 7 天 1 次 | 10:00 自动生成跟进提醒 |
| 沉睡客户 | 每 15 天 1 次 | 10:00 自动生成唤醒话术 |

---

## 三、跟进内容生成规则

### 3.1 内容轮换策略

每次跟进使用不同策略，避免重复：

```
第1次: 产品推荐 / 报价跟进
第2次: 成功案例 / 客户见证
第3次: 新品上市 / 库存特价
第4次: 行业资讯 / 季节推荐
第5次: 节日问候 / 限量优惠
```

### 3.2 个性化要素

每次跟进必须包含：
1. **客户称呼**（X总/X姐/X哥）
2. **历史记录引用**（"上次您问的XX面料..."）
3. **价值点**（质量/价格/服务/交期）
4. **行动号召**（给明确的下一步）

### 3.3 季节策略

| 季节 | 主推产品 | 话术方向 |
|------|---------|---------|
| 春季(3-5月) | 牛奶丝、汗布、T恤面料 | 夏装备货、早鸟价 |
| 夏季(6-8月) | 四面弹、轻薄罗纹 | 当季热销、现货充足 |
| 秋季(9-11月) | 摇粒绒、卫衣布、加绒罗纹 | 冬装备货 |
| 冬季(12-2月) | 加厚摇粒绒、双面呢替代 | 保暖面料、年底清仓 |

## 四、跟进状态标记

```
new         → 新线索，尚未联系
contacted   → 已联系，等待回复
negotiating → 报价中
sample_sent → 样品已寄送
trial_order → 试单中
repeat      → 复购客户
lost        → 流失
blacklist   → 黑名单
```

## 五、跟进自动化规则

### 5.1 自动提醒规则

- 每天 09:00: 生成当日跟进任务清单
- 每天 10:00: A 类客户跟进提醒
- 每天 10:30: B 类客户跟进提醒
- 每天 11:00: C 类客户跟进提醒
- 每 15 天: 沉睡客户唤醒提醒
- 实时: 客户回复即时提醒

### 5.2 自动升级/降级

- **自动升级**: B 类客户主动询价+回复积极 → 自动升级 A 类
- **自动降级**: A 类客户 7 天未回复 → 降 B 类；B 类 14 天未回复 → 降 C 类
- **自动休眠**: C 类 30 天未回复 → 标记"沉睡"
- **自动流失**: 180 天无互动 → 标记"流失"

### 5.3 无需跟进条件

- 客户明确表示不需要
- 客户已从其他渠道购买
- 客户要求不要打扰
- 客户被标记为黑名单
- 客户已流失超过 180 天
